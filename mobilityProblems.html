Mobile Networks and Distributed Algorithms

Papers

Questions about communication (routing) and mobility management in ad hoc (infrastructureless) mobile networks, as well as distributed algorithms (designed to run on network).

Self-stabilizing algorithms. These algorithms run on a network: these make no assumptions about initial configurations but are supposed to converge; each node has only local information and there is no global knowledge nor synchronisation. We want self-stabilizing algorithms for specific problems, and to start understanding the limits of such algorithms. Recently produced algorithms for several graph problems and established some comparisons between and limits of different models that are used.
